hkmu - Hackintosh maintenance utility

ABOUT

    A crude CLI tool for performing regular Hackintosh maintenance tasks;
    currently a work in progress.

INSTALL

    Place the program in a directory in your $PATH.

USAGE

    For usage information, see 'khmu help'.
